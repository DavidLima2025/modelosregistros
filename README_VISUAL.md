# Ajuste visual aplicado

A tela inicial foi ajustada para seguir o modelo aprovado:

- Nome: Assistente de Registros Policial.
- Função principal: Gerar Histórico.
- Cards centrais em destaque para os modelos de registro.
- Lateral direita com Históricos salvos, Favoritos, Ditar Relato, Relato Livre (IA) e Biblioteca Operacional.
- Menu esquerdo apenas como navegação de apoio.
- Cards principais abrem diretamente o formulário do modelo correspondente.

Observação: a imagem gerada é um mockup visual. O projeto foi ajustado para ficar o mais próximo possível dela dentro da estrutura React/Vite existente.
