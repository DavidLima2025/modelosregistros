# Correção aplicada

A versão anterior ainda podia abrir direto no formulário de Extravio. Esta versão força uma página inicial real:

- `activeTab` inicia como `inicio`.
- A página inicial é renderizada fora das abas de formulário.
- Os cards principais aparecem primeiro.
- O formulário só abre quando o usuário clica em um card.
- Sidebar com nome Assistente de Registros Policial.
